== Description ==

The CaptchaCO.IN Bitcoin Wallet is a secure cryptocurrency (BTC) wallet that allows you to send and receive bitcoins in your blog anytime and anywhere.

Your bitcoin wallet viewed and used right at your blog's dashboard, no need of logging into another website just to send and receive or even just view your bitcoin balance.

A Project Of: https://www.captchaco.in/

== Features ==
* Secure bitcoin wallet implementation
* Full control of wallet private key
* Bitcoin to USD exchange

A Project Of: https://www.captchaco.in/

== Installation ==

1. Upload `wp-bitcoin-wallet` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
